# TiMUS — The Simpler Way to Paper Trade

Practice trading with $100,000 in virtual cash. Real market data, real order types, free for everyone.

**Live site**: [spunshi24.github.io/TIMUS](https://spunshi24.github.io/TIMUS/)

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Flask + PostgreSQL backend (hosted on Railway)
- GitHub Pages for frontend deployment

## Local Development

```sh
git clone https://github.com/spunshi24/TIMUS.git
cd TIMUS/timus-main
npm install
npm run dev
```
